# Quiz — Project Context

## Purpose
Trivia game mobile app for Franq. Gamified learning experience
to help Personal Bankers understand financial products better.

## Folder map
- src/      → Application source code
- assets/   → Images, sounds, game assets
- docs/     → Game design docs, rules, product specs
- tests/    → Test suites

## Conventions
- Keep questions practical — real scenarios bankers face
- Portuguese language for all game content
- Difficulty levels: Iniciante, Intermediário, Avançado
